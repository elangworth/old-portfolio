About Me

My Work

Class Challenge on Search Engine Optimization

In this project I had to take starter code that contained mostly non-sematic HTML and change it to sematic HTML. I also consolidated the CSS code since the starter code had multiple duplications even though classes were already set up in the HTML. I made sure it also followed sematic structure. Some of the links in the starter HTML were not working and I fixed those as well. I learned that having semantic elements really helps increase a website's visibility and helps find the right customers for your business. Having great SEO helps search engines like Google find your site and put it at the top of the results page.
